jQuery-Knockout RequireJS Sample

This sample show how the plugin helps use KnockoutJS from jQuery using the jQuery-Knockout plugin

This is an AMD sample that uses RequireJS as its load library. This means that this sample has its code broken down into 3 files:

  • requirejs-config.js - A configuration file linked to from the HTML as the startup JavaScript that will be loaded by RequireJS. This file only config RequireJS and loads the application file.
  • app.js - The application file that uses the jQuery-Knockout plugin to bind the viewmodel to the HTML elements. This file is dependant on jQuery, the jQuery-Knockout-plugin and on the model file.
  • model.js - The model file that define the KnockoutJS view model that will be used in this sample (Very simple model). This file only depends on KnockoutJS.

Note that the dependancy of the plugin in jQuery and Knockout is dealt within the plugin itself.
