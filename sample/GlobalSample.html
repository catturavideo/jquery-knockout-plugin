jQuery-Knockout Global Sample

This sample show how the plugin helps use KnockoutJS from jQuery using the jQuery-Knockout plugin

This is a Global sample, this means that it uses the global objects exposed by jQuery and Knockout as global objects.

In this example a very simple model is instantiated twice, and each instance is bounded to 2 views using the plugin.
